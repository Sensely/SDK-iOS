Sensely SDK iOS

Consumers of the Sensely iOS SDK can leverage Sensely’s avatar, conversational platform, and Clinician’s Portal to create a solution that will gather data from patients and enable health providers to manage patient populations.

Prerequisites

  • iOS 10.0+
  • Xcode 8.0+ xcode
  • Cocoapods version 1.0 or later
  • iOS device

Quickstart

  • Add pod SenselySDK into your podfile
  • Run pod install to download and build Cocoapods dependencies.
  • Open the project by running open SenselySDKSample.xcworkspace.
  • Build and run the app.
