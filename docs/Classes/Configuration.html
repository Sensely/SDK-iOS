Configuration

@objc
public class Configuration : NSObject

Class responsible for handling all of the SDK Configuration options. Provides default values for Application-wide configuration properties. All properties are configurable via the respective setter method

You can provide configuration in your .plist file by adding the SenselyConfiguration key:

<key>SenselyConfiguration</key>
<dict>
<key>sdkLanguage</key>
<string>ja</string>
</dict>

Requires

To apply changes you must call Configuration.readFromInfoPlist() before pushing Avatar onto the screen
