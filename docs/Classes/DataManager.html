Data retrieval endpoints
  • gettingAssessments(completion:)

    Function for retrieving all assessments available for the currently logged in user At success you can retrieve assessments names and icons using the following functions:

    • DataManager.sharedInstance.stateMachine.getAssessmentIcons()
    • DataManager.sharedInstance.stateMachine.getAssessmentNames()

    You can get procedure id of an assessment DataManager.sharedInstance.stateMachine.getProcedureId(...)

    Also you may get avatar code and voice recogntion language:

  • DataManager.sharedInstance.stateMachine.getAvatarCode()

  • DataManager.sharedInstance.stateMachine.getSpeechInputLanguage()

    • Precondition

    should be called after authorization endpoints such as senselyLogin
